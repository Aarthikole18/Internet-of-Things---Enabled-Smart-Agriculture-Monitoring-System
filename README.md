# 🌱 IoT Smart Agriculture Monitoring System (Streamlit Simulation)

## 🚀 Overview
This project is a simulation-based Smart Agriculture IoT system built using Python and Streamlit. It demonstrates how environmental sensor data can be monitored and used for automated irrigation decisions in smart farming.

The system simulates real-time data for temperature, humidity, soil moisture, and light intensity, and uses simple threshold logic to control a virtual water pump.

---

## 📊 Features
- 🌡 Temperature monitoring (simulated)
- 💧 Humidity tracking
- 🌱 Soil moisture analysis
- 💡 Light intensity monitoring
- 🚰 Automatic irrigation decision (Pump ON/OFF logic)
- 📈 Live Streamlit dashboard
- 📊 Real-time charts and visualization
- 🧠 Threshold-based decision system

---

## 🧠 Tech Stack
- Python
- Streamlit
- Pandas
- Random module

Sensor Data Generation → Simulation Engine → Threshold Logic → Pump Decision → Data Storage → Streamlit Dashboard Visualization
